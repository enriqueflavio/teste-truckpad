# truckpad (truckpad)

Teste para Truckpad

## Instalando as dependencias
```bash
npm install
```

## Instalando o quasar
```bash
npm install -g @quasar/cli
```

### Pra rodar em modo de desenvolvimento
```bash
quasar dev
```

### Buildar para produção
```bash
quasar build
```
